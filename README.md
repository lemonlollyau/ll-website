# LemonLolly website

Static marketing site for LemonLolly (Adelaide marketing studio).
Hosted on Vercel, source of truth on GitHub. Push to `main` = publish.

- `index.html` — home page
- `favicon.svg` — site icon
- `vercel.json` — hosting config (clean URLs)
- `robots.txt`, `sitemap.xml` — SEO

Built from the LemonLolly Home design in Cowork.
